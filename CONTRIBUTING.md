# Contributing to Space Arena

Thank you for contributing to Space Arena.

This project is an open-source real-time multiplayer browser game built for learning, collaboration, and experimentation.

---

## Ways to Contribute

You can contribute by:

- Fixing bugs
- Improving UI/UX
- Adding sound effects
- Improving performance
- Adding visual effects
- Improving mobile support
- Refactoring systems
- Improving documentation

---

## Project Setup

### Install dependencies

```bash
npm install
```

### Build shared package

```bash
cd shared
npm run build
```

### Start backend

```bash
cd server
npm run dev
```

### Start frontend

```bash
cd client
npm run dev
```

---

## Contribution Guidelines

- Keep code modular and readable
- Avoid unnecessary complexity
- Follow existing project structure
- Test features before opening PR
- Keep commits focused and meaningful

---

## Beginner Friendly Contributions

Some good beginner-friendly areas:

- UI polish
- Sound effects
- Small gameplay improvements
- Bug fixes
- Responsive improvements
- Particle effects

---

## Pull Requests

When opening a PR:

- Clearly describe the change
- Keep PRs reasonably small
- Add screenshots/GIFs if UI changed
- Mention related issue if applicable
